## 2 简介

本文档介绍如果你在使用过程中发现问题，本文档将帮助你进行反馈，欢迎使用 Lierda LTE-EC71X OpenCPU SDK！

## 3 项目概述

Lierda LTE-EC71X OpenCPU SDK 是利尔达物联网推出的 EC71X 系列模组的 OpenCPU 开发套件。该 SDK 允许开发者直接在模组上运行自定义应用程序，无需外挂 MCU。

**主要特性：**

| 特性 | 说明 |
| ---- | ---- |
| 芯片支持 | F6B_A、F6D_A、F7B_A、K2B_A、K2F_A 等系列 |
| 操作系统 | FreeRTOS 实时操作系统 |
| API 标准 | CMSIS-RTOS2 标准 API |
| 中间件 | MQTT、HTTP、LwIP、mbedTLS、LittleFS 等 |
| 编译工具 | GCC、Keil |
| 系统接口 | 文件系统、GPIO、UART、低功耗管理等 |

## 4 问题反馈

### 4.1 反馈类型

| 类型 | 说明 | 示例 |
| ---- | ---- | ---- |
| Bug 报告 | 功能异常、崩溃、错误等 | 编译失败、运行崩溃、功能不符合预期 |
| 功能建议 | 新功能需求、现有功能改进 | 希望增加 XX 协议支持、优化 XX 接口性能 |
| 文档问题 | 文档错误、缺失、不清晰 | 某 API 说明有误、缺少使用示例 |
| 使用疑问 | 使用过程中的问题咨询 | 如何实现 XX 功能、某参数的含义 |

### 4.2 反馈模板

请使用以下模板提交问题：

1. **问题类型**
   Bug 报告 / 功能建议 / 文档问题 / 使用疑问（请选择一项）
2. **问题描述**
   清晰描述问题现象
3. **环境信息：**
   SDK 版本：
   芯片型号：F6B_A/F6D_A/F7B_A/K2B_A/K2F_A（请选择一种或多种）
   编译工具链版本：
   操作系统：
4. **复现步骤**
   详细的复现步骤
5. **日志输出**
   编译日志或运行日志
6. **期望行为**
   说明期望的正确行为

### 4.3 功能建议补充说明

提交功能建议时，请额外说明：

- 功能需求场景
- 期望的实现方式
- 与其他功能的兼容性考虑
