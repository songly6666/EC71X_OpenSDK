利尔达 LTE Cat.1 bis 模组产品介绍
====================================


.. note::

   本文档用于介绍利尔达 LTE Cat.1 bis 系列模组的产品定位、核心产品矩阵、技术参数、技术优势、行业应用及技术支持能力。

一、产品概述
------------

利尔达 LTE Cat.1 bis 系列是专为电池供电、小型化物联网终端打造的高性能 4G 通信模组。基于移芯通信最新一代 EC718 / 718PM 低功耗平台，实现了 10×13mm 最小尺寸和 1.7mm 极致厚度，功耗表现媲美 NB-IoT，同时具备工业级可靠性和全球覆盖能力。

本系列已形成覆盖标准超薄、宽压供电、全网通、全球多频段、集成定位的完整产品矩阵，广泛应用于智能穿戴、智能表计、定位追踪、安防传感及工业互联等领域，帮助客户快速实现产品小型化、低功耗化和全球化部署。

.. note:: 核心客户价值

   - **极致尺寸**：1.7mm 超薄设计，突破传统 Cat.1 模组结构限制。
   - **超长续航**：718PM 平台深度优化，电池寿命提升 30% 以上。
   - **快速上市**：统一封装焊盘与软件 API，开发周期缩短 50%。
   - **全球部署**：多频段版本与完整国际认证，提供一站式出海解决方案。

二、核心产品矩阵
----------------

NT26F6D0 国内全网通版
^^^^^^^^^^^^^^^^^^^^^

**产品定位：** 国内市场主力通用型 Cat.1 bis 模组。

**核心规格：**

- 核心平台：移芯 EC718
- 尺寸规格：17.7×15.8×2.4mm，可选 1.7mm 超薄款
- 封装类型：LGA
- 网络制式：国内三大运营商 4G 全网通

**核心卖点：**

- 极致超薄设计：1.7mm 厚度，适配结构空间受限设备。
- 超低功耗：休眠功耗低至 3μA，显著提升电池续航。
- 丰富功能：支持 FOTA 远程升级、VoLTE 高清语音、WiFi-Scan 辅助定位。
- OpenCPU 友好：提供完整 SDK，支持客户应用直接运行在模组上。

**推荐场景：** 智能穿戴、共享设备、OpenCPU 开发、消费级物联网终端。

NT26F6D0 全球多频段版
^^^^^^^^^^^^^^^^^^^^^

**产品定位：** 海外市场一站式解决方案。

**核心规格：**

- 核心平台：移芯 EC718
- 尺寸规格：17.7×15.8×2.4mm，可选 1.7mm 超薄款
- 封装类型：LGA
- 版本划分：

  - NT26F6D0-GL：全球通用版
  - NT26F6D0-EU：欧洲版
  - NT26F6D0-LA：拉美版
  - NT26F6D0-NA：北美版
  - NT26FJPD70NNA：日本版，带 TELEC 认证

**核心卖点：**

- 全球频段覆盖：支持全球主流 LTE Cat.1 频段，无需额外硬件调整。
- 完整国际认证：自带 CE / FCC / Anatel / TELEC 等认证，缩短海外上市周期。
- 功耗优化：内置 APT 智能功耗调节技术，相比标准版功耗降低 20%。
- 无缝切换：统一软件 API 接口，国内 / 海外版本代码兼容。

**推荐场景：** 出海产品、跨境共享设备、海外民用物联网终端。

NT26K2E0 全球定位版
^^^^^^^^^^^^^^^^^^^

**产品定位：** 集成 GNSS 的全球追踪专用模组。

**核心规格：**

- 核心平台：移芯 EC718
- 尺寸规格：17.7×15.8×2.4mm
- 封装类型：LGA
- 定位能力：集成 GPS / 北斗双模 GNSS

**核心卖点：**

- 一体化设计：Cat.1 + GNSS 二合一，无需额外定位芯片，节省 BOM 成本。
- 全球部署：支持多频段切换，自带完整国际认证。
- eSIM 支持：支持 eSIM 和远程 SIM 配置，便于全球批量部署。
- 统一软件：与全系列产品软件兼容，降低开发维护成本。

**推荐场景：** 出海资产追踪、物流监控、宠物定位、人员工牌。

NT26F6C0 国内宽压工业版
^^^^^^^^^^^^^^^^^^^^^^^

**产品定位：** 工业级宽温宽压专用模组。

**核心规格：**

- 核心平台：移芯 EC718
- 尺寸规格：17.7×15.8×2.4mm
- 封装类型：LCC + LGA 加固封装
- 供电范围：2.3V~4.5V 超宽电压
- 工作温度：-40℃~+85℃ 工业级宽温

**核心卖点：**

- 超宽压供电：支持电池低压跌落和太阳能混合供电。
- 工业级可靠性：-40℃~+85℃ 宽温运行，适应极端户外环境。
- 加固封装：LCC + LGA 双封装，贴片更稳固，抗振动能力强。
- 高稳定性：抗电源波动能力强，适合无人值守设备。

**推荐场景：** 无线表计（水 / 电 / 气）、烟感报警器、户外传感终端、工业 DTU。

四、核心技术优势深度解析
------------------------

1. 极致小型化设计
^^^^^^^^^^^^^^^^^

采用先进的系统级封装（SiP）技术，在 17.7×15.8mm 的小尺寸内集成完整 4G 通信系统。1.7mm 极致厚度比传统 Cat.1 模组薄 30% 以上，为穿戴设备和小型终端提供了宝贵结构空间，最小可做到 10×13mm 尺寸。

2. 行业领先的低功耗
^^^^^^^^^^^^^^^^^^^

基于移芯 EC718 / 718PM 平台深度功耗优化，NT26 系列休眠功耗低至 3μA，工作功耗比同类产品降低 25%，电池供电设备续航时间可提升 30% 以上，真正实现“一次安装，多年使用”。

3. 工业级可靠性
^^^^^^^^^^^^^^^

NT26F6C0 支持 -40℃~+85℃ 工业级宽温范围和 2.3V~4.5V 超宽电压输入，能够在极端温度和不稳定供电环境下稳定运行，特别适合户外表计和工业传感应用。

4. 统一封装与软件
^^^^^^^^^^^^^^^^^

全系列产品采用统一的 LCC / LGA 封装和软件 API 接口，客户可以在不同型号之间无缝切换，减少 PCB 设计和应用代码修改，降低开发成本和产品迭代周期。

5. 丰富的增值功能
^^^^^^^^^^^^^^^^^

支持 OpenCPU 开发模式，客户可以直接在模组上运行应用程序，无需额外 MCU。内置 TTS 语音播报、扫码算法、WiFi-Scan 定位等功能，进一步降低 BOM 成本。

6. 全球化部署能力
^^^^^^^^^^^^^^^^^

提供覆盖全球、欧洲、北美、拉美、日本等主要市场的多频段版本，自带完整国际认证，帮助客户快速进入海外市场，实现“一个设计，全球销售”。

五、行业解决方案
----------------

智能穿戴
^^^^^^^^

1.7mm 超薄设计 + 3μA 超低功耗，适配智能手表、医疗手环、定位工牌等穿戴设备，提供更长续航时间和更舒适的佩戴体验。

智能表计
^^^^^^^^

2.3V~4.5V 宽压输入 + 工业级宽温，支持电池和太阳能供电，适用于水表、气表、电表等无源表计，实现长周期免维护运行。

定位追踪
^^^^^^^^

内置 WiFi-Scan 辅助定位 + GNSS 双模定位 + 全球多频段支持，适用于人员定位、宠物追踪、物流资产监控等场景，提供精准可靠的位置服务。

安防传感
^^^^^^^^

超低功耗 + 高可靠性，适用于烟感报警器、燃气探测器、入侵检测等安防设备，确保在紧急情况下能够及时报警。

金融支付
^^^^^^^^

内置 TTS 和扫码算法 + 双 SIM 卡支持，适用于云喇叭、迷你 POS、扫码设备等金融终端，提供稳定可靠的支付通信。

共享经济
^^^^^^^^

全球多频段版本 + 完整国际认证，适用于共享单车、共享充电宝、共享雨伞等共享设备，支持全球范围部署和运营。
