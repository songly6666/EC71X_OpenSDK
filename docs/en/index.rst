Lierda LTE Cat.1 bis Module Product Introduction
==================================================


.. note::

   This document introduces the product positioning, core product matrix, technical specifications, technical advantages, industry applications, and technical support capabilities of the Lierda LTE Cat.1 bis series modules.

1. Product Overview
-------------------

The Lierda LTE Cat.1 bis series is a high-performance 4G communication module designed specifically for battery-powered, miniaturized IoT terminals. Based on EigenComm's latest-generation EC718 / 718PM low-power platform, it achieves a minimum size of 10x13mm and an ultra-thin thickness of 1.7mm, with power consumption comparable to NB-IoT while maintaining industrial-grade reliability and global coverage.

This series has formed a complete product matrix covering standard ultra-thin, wide-voltage supply, full-band domestic, global multi-band, and integrated positioning variants. It is widely used in smart wearables, smart metering, location tracking, security sensing, and industrial IoT, helping customers quickly achieve product miniaturization, low power consumption, and global deployment.

.. note:: Core Customer Value

   - **Ultimate Size**: 1.7mm ultra-thin design, breaking through traditional Cat.1 module structural limitations.
   - **Extended Battery Life**: Deep optimization on the 718PM platform, improving battery life by over 30%.
   - **Fast Time-to-Market**: Unified package footprint and software APIs, reducing development cycles by 50%.
   - **Global Deployment**: Multi-band versions with complete international certifications, providing one-stop overseas solutions.

2. Core Product Matrix
----------------------

NT26F6D0 Domestic Full-Band Version
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**Product Positioning:** Mainstream general-purpose Cat.1 bis module for the domestic market.

**Core Specifications:**

- Core Platform: EigenComm EC718
- Dimensions: 17.7x15.8x2.4mm, optional 1.7mm ultra-thin variant
- Package Type: LGA
- Network: China tri-carrier 4G full-band

**Key Advantages:**

- Ultra-thin design: 1.7mm thickness, suitable for space-constrained devices.
- Ultra-low power: Sleep current as low as 3uA, significantly extending battery life.
- Rich features: Supports FOTA remote upgrade, VoLTE HD voice, WiFi-Scan assisted positioning.
- OpenCPU friendly: Provides complete SDK, supporting customer applications running directly on the module.

**Recommended Scenarios:** Smart wearables, shared devices, OpenCPU development, consumer IoT terminals.

4. Core Technical Advantages
-----------------------------

1. Ultimate Miniaturization
^^^^^^^^^^^^^^^^^^^^^^^^^^^

Using advanced System-in-Package (SiP) technology, a complete 4G communication system is integrated within the 17.7x15.8mm footprint. The 1.7mm ultra-thin thickness is over 30% thinner than traditional Cat.1 modules, providing valuable structural space for wearable devices and small terminals, with minimum size achievable at 10x13mm.

2. Industry-Leading Low Power
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Based on deep power optimization of the EigenComm EC718 / 718PM platform, the NT26 series achieves sleep current as low as 3uA, with working power consumption 25% lower than similar products. Battery-powered device battery life can be improved by over 30%, truly achieving "install once, use for years".

3. Industrial-Grade Reliability
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

The NT26F6C0 supports -40C~+85C industrial-grade wide temperature range and 2.3V~4.5V ultra-wide voltage input, capable of stable operation in extreme temperatures and unstable power supply environments, particularly suitable for outdoor metering and industrial sensing applications.

4. Unified Package and Software
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

The entire product series uses unified LCC / LGA packages and software API interfaces. Customers can seamlessly switch between different models, reducing PCB design and application code modifications, lowering development costs and product iteration cycles.

5. Rich Value-Added Features
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Supports OpenCPU development mode, allowing customers to run applications directly on the module without an additional MCU. Built-in TTS voice broadcast, barcode scanning algorithms, WiFi-Scan positioning and other features further reduce BOM costs.

6. Global Deployment Capability
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Provides multi-band versions covering global, European, North American, Latin American, Japanese and other major markets, with complete international certifications built-in, helping customers quickly enter overseas markets and achieve "one design, global sales".

2.3V~4.5V wide voltage input + industrial-grade wide temperature, supporting battery and solar power supply, applicable to water meters, gas meters, electricity meters and other passive meters, achieving long-cycle maintenance-free operation.

Location Tracking

Built-in WiFi-Scan assisted positioning + GNSS dual-mode positioning + global multi-band support, applicable to personnel positioning, pet tracking, logistics asset monitoring and other scenarios, providing accurate and reliable location services.

Sharing Economy
